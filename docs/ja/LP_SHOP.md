## Proof of Physical Evidence（PoPE）

MORM Shopの心臓部。**世界初の商用シップ**。

### 仕組み

```
[セラー]                       [バイヤー]
   │                             │
   ├─ MORM専用カメラで梱包動画撮影 │
   │  （ブロックハッシュ透かし）   │
   ├─ 動画ハッシュをチェーン記録    │
   ├─ 商品発送                    │
   │                             │
   │                             ├─ 受取
   │                             ├─ MORM専用カメラで開封動画撮影
   │                             │  （ブロックハッシュ透かし）
   │                             └─ 動画ハッシュをチェーン記録
   ↓                             ↓
   ┌──────────────────────────────┐
   │ AI / Validator による整合性検証 │
   │ ・動体解析                     │
   │ ・コンテンツ照合               │
   │ ・タイムスタンプ整合性         │
   └──────────────────────────────┘
            ↓                ↓
      [一致] エスクロー解除   [不一致] 自動Refund + Node-Lock
```

### なぜ詐欺ができないのか

1. **ブロックハッシュ透かし** — 動画に「撮影瞬間のチェーン状態」が焼き込まれる。後から動画を作り直しても、ハッシュは絶対に一致しない
2. **専用カメラ必須** — 撮影アプリが偽造を弾く。AI生成動画も検知される
3. **動体解析AI** — 不自然なカット、ループ、合成は瞬時に検出
4. **エスクロー** — 99%の資金がスマコンにロック。詐欺確定なら自動Refund

## FAQ

### Q. 動画撮影が面倒ではないですか？

A. 専用カメラアプリは「録画開始 → 開封 → 録画停止」の30秒で完了します。これだけで100%詐欺保護されると考えれば、十分価値ある工程です。

### Q. 開封動画を撮り忘れたら？

A. 異議申立はできなくなります。動画なし=「商品到着OK」と見なされ、自動でエスクロー解除されます。

### Q. プライバシーは大丈夫？

A. 梱包・開封動画は取引完了後、低コストアーカイブ層へ移動し、第三者から閲覧不可になります。証拠としてのハッシュのみがチェーンに残ります。

### Q. 物流会社のミスや破損は？

A. 物流追跡情報もチェーン記録対象。物流側のミスは出品者・購入者・物流会社の3者で仲裁します。MORM Shopは保険オプションも提供予定。

### Q. デジタル財の場合は？

A. 動画撮影は不要。代わりにGeneration IDまたはライセンスキー暗号化転送で証明します。エスクローは同様に動作します。

### Q. 高額商品（数百万円〜）は？

A. 高額取引には追加バリデーター仲裁、保険オプション、物流時の写真撮影マイルストーンを提供予定（MORM Phase 1以降）。
